2. Description of the Application

Signalo is a native Android application that enables users to monitor live network stats of their device. The main purpose is to view the signal strength (dBm) for both cellular and Wi-Fi networks in a gauge graph.

3. Android Permissions and Device Data

To access live cellular and Wi-Fi signal strength, Signalo requires specific Android network and location permissions. These permissions are strictly mandated by the Android operating system to read network states. Signalo uses these permissions solely to display the signal gauge. The application does not derive, track, store, or transmit your physical location or network identifiers.

4. Data Collection and Sharing

Signalo does not collect, store, process, or share any personal data. All network information requested from Android is read locally in memory and is instantly discarded. It is never transmitted to the developer, any server, or any third party.

5. User Rights (GDPR)

Users possess rights under the General Data Protection Regulation (GDPR), including the right of access, rectification, and erasure. However, because Signalo does not collect or retain any personal data, requests to access, modify, or delete personal data cannot be fulfilled, as no such data exists.

General inquiries can be sent to the contact address listed above.
