Welcome to Signalo

Signalo is a native Android application that enables users to monitor live network stats of their device. The main purpose is to view the signal strength (dBm) for both cellular and Wi-Fi networks in a gauge graph.

Features

  • Live signal strength visualization (dBm) via an intuitive gauge graph with color-coded thresholds
  • Detailed cellular stats: Network Provider, Network Type, Frequency Band, Cell ID (ECI)
  • Detailed Wifi stats: SSID, Frequency Band, Encryption, Linkspeed
  • Dual-SIM support with per-SIM signal comparison
  • Manual Wifi refresh for the newest and most precise signal values
  • Automatic no-SIM detection with fallback to Wifi page
